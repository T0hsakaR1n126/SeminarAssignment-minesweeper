Minesweeper Running Guide

1.Use the guide in exercise to install OCaml and dune.
2.Run ```dune build``` to build.
3.Run ```dune exec ./main.exe -- --rows 8 --cols 8 --mines 10 --seed 42``` to open the game(Use command 'cd' to change to the directory the main.exe locates). Parameters of gameboard can be changed.
